# Simple Python Web Scraper CLI

A lightweight, terminal-based tool to extract structured data from websites and save it to a CSV file.

## Features
- Custom URL input.
- Adjustable CSS selectors for different websites.
- Handles missing data gracefully.
- Adds timestamps to each entry.
- Exports data using Pandas.
